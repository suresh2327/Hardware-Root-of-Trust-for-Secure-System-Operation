# 📊 05 Presentations & Documentation

This directory contains the slide decks and final presentations that summarize the entire **Hardware Root-of-Trust (RoT)** project for both technical and non-technical stakeholders.

## 📂 Included Materials
- **`HW_RootOfTrust_Final.pptx`**: The comprehensive final presentation detailing the motivation, architecture, RTL design strategies, and UVM verification results.
- **`TeamPresentation.pptx`**: The team's collaborative slide deck, emphasizing individual contributions, workflows, and the overall success of the project.

## 🎯 Purpose
These documents serve as the definitive high-level overview of the ASIC design lifecycle applied in this project. They are intended for use during project defenses, portfolio showcases, and technical AI reviews.
